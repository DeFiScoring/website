# DeFiScoring API Documentation (v1.0)

Welcome to the DeFiScoring API. Our API provides programmatic access to real-time risk metrics, AI-driven security narratives, and on-chain health scores.

## 1\. General Information

  * **Base URL:** `https://api.defiscoring.com/v1`
  * **Format:** All responses are returned in `application/json`.
  * **Rate Limits:** 100 requests per minute (Free Tier). For higher limits, please contact `api@defiscoring.com`.

-----

## 2\. Authentication

Currently, public endpoints do not require an API key. However, for the **AI Insights** and **Personalized Health** endpoints, you must include your API key in the request header:

`X-API-Key: YOUR_API_KEY`

-----

## 3\. Endpoints

### A. Get Protocol Risk Score

Returns the current aggregate risk score and pillar breakdown for a specific protocol.

  * **Endpoint:** `/score/{protocol_slug}`
  * **Method:** `GET`
  * **Example Request:** `GET /v1/score/uniswap-v3`

**Success Response:**

```json
{
  "protocol": "Uniswap V3",
  "slug": "uniswap-v3",
  "timestamp": "2026-04-20T16:00:00Z",
  "aggregate_score": 94,
  "rating": "AAA",
  "pillars": {
    "security": 98,
    "decentralization": 92,
    "economics": 95,
    "maturity": 90
  },
  "audit_count": 4,
  "last_audit_date": "2025-11-12"
}
```

-----

### B. Get AI-Powered Security Assessment

Returns a narrative summary generated by our AI Auditor regarding a specific contract.

  * **Endpoint:** `/ai-assessment/{contract_address}`
  * **Method:** `GET`
  * **Example Request:** `GET /v1/ai-assessment/0x1f9840...`

**Success Response:**

```json
{
  "address": "0x1f9840...",
  "ai_model": "Claude-3.5-Sonnet-DeFi",
  "risk_summary": "Centralized admin keys detected with no timelock. High risk of parameter manipulation.",
  "red_flags": ["No-Timelock", "Single-Owner-Admin"],
  "score_impact": -25
}
```

-----

### C. Get User Health Score

Analyzes a wallet's on-chain history to return a "DeFi Credit Score."

  * **Endpoint:** `/health/{wallet_address}`
  * **Method:** `GET`
  * **Example Request:** `GET /v1/health/0x71C765...`

**Success Response:**

```json
{
  "wallet": "0x71C765...",
  "health_score": 720,
  "status": "Healthy",
  "metrics": {
    "loan_reliability": "High",
    "gov_participation": 12,
    "liquidity_provision_duration": "450 days"
  },
  "alerts": [
    {
      "type": "Risk-Exposure",
      "message": "You have $500 in a protocol with a score < 40."
    }
  ]
}
```

## 4\. Error Codes

| Code | Name | Description |
| :--- | :--- | :--- |
| **400** | Bad Request | Missing parameters or invalid address format. |
| **401** | Unauthorized | Invalid or missing API key. |
| **404** | Not Found | Protocol or contract not yet indexed in our database. |
| **429** | Rate Limit | You have exceeded the 100 req/min limit. |

-----

## 5\. Integration Example (JavaScript)

```javascript
const fetchScore = async (slug) => {
  try {
    const response = await fetch(`https://api.defiscoring.com/v1/score/${slug}`, {
      headers: { 'X-API-Key': 'your_key_here' }
    });
    const data = await response.json();
    console.log(`The score for ${data.protocol} is ${data.aggregate_score}`);
  } catch (error) {
    console.error("Error fetching DeFiScoring data:", error);
  }
};
```

## 6\. Webhooks (Pro Feature)

Stay updated in real-time. We can push data to your server whenever a protocol's score drops by more than 10 points.

  * **Payload:** Includes `protocol_slug`, `old_score`, `new_score`, and `reason`.
